The ruleset is as follows:
If the argument is evenly divisible by 3, return “Fizz”.
If the argument is evenly divisible by 5, return “Buzz”.
If the argument matches both rules, return “FizzBuzz”.

Input || Output
|-----||-------|
 1    ||
 2    ||
 3    || Fizz
 4    ||
 5    || Buzz

To demonstrate that your function works, run your function through a loop that supplies 1 through 100 as an argument for your function.
